# Android Native Client (Java/Kotlin)

Native Android implementation of the encrypted messenger app using Java/Kotlin with WebSocket integration.

## Setup

1. Open in Android Studio
2. Configure your API endpoint in `Config.java`
3. Build and run

## Features

- Native Android UI
- WebSocket real-time communication
- End-to-end encryption
- Secure key storage using Android Keystore

## Dependencies

- OkHttp for WebSocket
- Retrofit for REST API
- Gson for JSON parsing
- Android Keystore for secure key storage
